# Welcome to Fastlytics 🏎️💨

> **Democratising F1 telemetry data for everyone.**

We are an open-source community dedicated to making Formula 1 telemetry, race analytics, and historical performance data accessible, transparent, and easy to understand for fans and developers alike.

## 🌟 Our Mission

Formula 1 produces millions of data points every race weekend. Traditionally, this data has been locked behind expensive enterprise tools or intimidating APIs. Fastlytics exists to break down those barriers by providing high-quality, open-source visualization tools and analytics platforms that anyone can easily deploy and use.

## 🚀 Projects

- **[Fastlytics Dashboard](https://github.com/Fastlytics/Fastlytics)**: Our flagship React application for granular F1 telemetry analysis, real-time race data, and historical performance comparisons. 
- *(More projects coming soon!)*

## 🛠️ Technology Stack

We build modern, fast, and scalable applications using:
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Visualization**: Recharts, shadcn/ui
- **Backend/Infrastructure**: Cloudflare Workers, Supabase
- **Data**: OpenF1 API
